# Clinical-Anesthesia-QA-System-using-RAG-and-LLMs
An AI assistant for anesthesia knoweldge using retrieval-augmented generation and LLMs to answer clinical questions

The goal is to make an intelligent system that could help anesthesiologist to query clinical questions releated to anesthesia and retrieve accurate and context aware answers.
