# Images Directory

Place your images here for use in the README.

## How to Add Images to README

1. **Copy your images** to this directory (`docs/images/`)
2. **Use this markdown syntax** in README.md:

```markdown
![Description of image](docs/images/your-image-name.png)
```

Or with a link:

```markdown
[![Description of image](docs/images/your-image-name.png)](https://optional-link.com)
```

## Supported Formats

- `.png` (recommended for screenshots)
- `.jpg` / `.jpeg` (for photos)
- `.gif` (for animated images)
- `.svg` (for vector graphics)
